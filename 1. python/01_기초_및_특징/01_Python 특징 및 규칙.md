	인터프리터 언어(Interpreter Langue)
		- 작동 방식 : 줄 단위로 즉시 읽어서 실행하는 스크립트 언어
		- 코드 수정과 테스트가 매우 빠르다.
			
	동적 타입 언어(Dynamically Typed Language)
		-실행 시점에 변수의 자료형이 결정된다.
		- 자료형을 미리 선언할 필요가 없어 코드가 간결하고 유연하다.
			
	플랫폼 독립적 언어 (Platform Independent)
		- OS에 상관없이(윈도우, 리눅스, 맥 등) 어디서나 동일하게 작동.

	파이썬 키워드
		- 파이썬에서 예약된 단어
		- 변수명, 함수명 또는 식별자로 사용할 수 없다.
		- 파이썬 키워드

![[Python keywords.png]]

	파이썬 식별자, 변수이름 규칙
		- 소문자, 대문자,숫자,_ 로 조합 가능하다.
		- 숫자로 시작할 수 없다.
		- 키워드를 식별자로 사용할 수 없다.
		- '_' 을 제외한 특수문자는 사용할 수 없다.
		- 식별자의 길이 제한이 없다. ※ 적당한 길이 사용

### 명령문 (Statement)
	프로그램에서 실행되는 기본 단위

	명령문
```
a = 1
```

	여러 줄 명령문 (Multi-line Statement)
		- 자료형으로 처리 : 괄호 (), 대괄호 [], 중괄호 {}
		- 역슬래시(\)
		- 세미콜론(;) 기호로 다중 명령문 가능
```
자료형으로 처리
a = (1 + 2 + 3 +  4 + 5 + 6 + 7 + 8 + 9)

역슬래시(\)
a = 1 + 2 + 3 + \  
	4 + 5 + 6 + \
	7 + 8 + 9
	
세미콜론(;) 기호로 다중 명령문
a = 1; b = 2; c = 3;
```

### 들여쓰기 (Indentation)
		- 파이썬은 {} 대신 들여쓰기로 블록을 정의.
		- 일반적으로 4칸을 사용. ※ Tab 사용
```
ex)
for i in range(1,11) :
	print(i)
	if i == 5:
		break
		
	→ 1
	  2 
	  3 
	  4 
	  5
```

### Docstring
		- 함수(function), 방법(method), 클래스(class), 모듈(module)
		   을 만들 때, 그 바로 아래에 설명서로 붙여주는 문자열 리터럴(literals)
		- "__doc__" 속성(attribute)으로 객체와 연결된다.
		- 함수,방법,클래스,모듈 등의 설명서 느낌
