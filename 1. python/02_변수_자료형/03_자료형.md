	Python 프로그래밍의 모든것은 객체이다.
	데이터형(Data types) : 실제로는 class 이며, 변수는 해당 클래스의 인스턴스 객체이다. 
	종류 : 
### 숫자 형(number)

	- 정수 (int): 메모리가 가용 가능한 범위 내에서 길이 제한이 없다.
```
#정수형 확인

a = 5 
print(a, "is of type", type(a))

-> 5 is of type <class 'int'>
```

	- 실수 (float): 소수점 이하 15자리까지 정확하다.
	    float : 소수점 이하 15자리까지 정확도 보장
```
# 실수형 확인

a = 2.0 
print(a, "is of type", type(a))

-> 2.0 is of type <class 'float'>
```

	- 복소수 (complex): 실수부와 허수부(`j`)로 구성된다.
		isinstance() : 주어진 값이 특정 클래스에 속하는지를 확인 (True 또는 False)
```
# 복소수 확인 및 판별

a = 1 + 2j
print(a, "is complex number?", isinstance(1+2j, complex))


-> (1+2j) is complex number? True
```

	- 형변환 (Type Conversion)
		int(), float(), complex() 함수를 사용하여 자료형을 강제로 변환할 수 있다.
		int와 float 간의 연산은 결과가 float로 강제 변환

	- 소수 모듈(decimal) : 유한한 자리수까지 절삭되어 저장
```
# decimal

import decimal

print(decimal.Decimal('1.1') + decimal.Decimal('2.2'))

-> 3.3
```

	- 분수 모듈(fraction) : 유리수를 분수 형태로 다룰 때 사용
```
# fractions

import fractions

print(fractions.Fraction(1.5))
print(fractions.Fraction(5))
print(fractions.Fraction(1, 3))

-> 3/2
   5
   1/3
```

	- 수학 모듈(math)
		기본 연산자로 하기 힘든 복잡한 수학 계산(삼각함수, 로그, 제곱근 등)을 모아놓은 도구함
		원주율(pi)이나 자연상수(e) 같은 수학적 상수도 포함

	- 랜덤 모듈(random)
		무작위 숫자를 생성하거나, 목록에서 무작위로 하나를 뽑을 때 사용하는 도구함

### 문자 형(string)
	유니코드(Unicode) 문자들의 나열, 한 번 생성된 문자열은 내부의 문자를 직접 변경할 수 없다.
	선언: 작은따옴표(')나 큰따옴표(")를 사용하며, 여러 줄은 ''' ''' 또는 """ """를 사용.
	
	- 이스케이프 시퀀스 (Escape Sequences)
		문자열 내에서 특수한 기능을 수행

```
\n  # 줄 바꿈 (Newline)

\t  # 탭 (Tab)

\\  # 백슬래시 출력

\'  # 작은따옴표 출력

\"  # 큰따옴표 출력

\b  # 백스페이스

\r  # 캐리지 리턴

\v  # 수직 탭

\f  # 폼 피드

\ooo  # 8진수 문자

\xHH  # 16진수 문자

```
