## 함수 기본

### 함수(Function)

	- 특정 작업을 수행하기 위한 명령문의 묶음이다.
	- 프로그램을 작은 단위로 나눌 수 있다.
	- 코드 중복을 줄이고 재사용성을 높인다.

### 함수 선언 구조

```python
def 함수이름(매개변수):
    실행문
    return 반환값
```

### 함수 예제

```python
def greet(name):
    """사람 이름을 받아 인사하는 함수"""
    print("Hello, " + name + ". Good morning!")


greet("Kim")
```

### return

	- 함수 실행 결과를 호출한 곳으로 돌려준다.

```python
def add(a, b):
    return a + b

result = add(3, 5)
print(result)
```
