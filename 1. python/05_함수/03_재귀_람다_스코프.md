## 재귀, 람다, 스코프

### 재귀(Recursion)

	- 함수가 자기 자신을 다시 호출하는 것이다.
	- 종료 조건이 반드시 있어야 한다.

	팩토리얼 예제

```python
def factorial(x):
    if x == 1:
        return 1
    else:
        return x * factorial(x - 1)

print(factorial(5))
```

	실행 결과

```text
120
```

	주의사항

```text
종료 조건이 없으면 무한 호출 또는 RecursionError가 발생한다.
```

### 람다 함수(Lambda Function)

	- 이름 없이 간단한 함수를 만들 때 사용한다.
	- 하나의 표현식만 사용할 수 있다.

	기본 구조

```python
lambda arguments: expression
```

	예제

```python
double = lambda x: x * 2
print(double(5))
```

	filter와 함께 사용

```python
my_list = [1, 5, 4, 6, 8, 11, 3, 12]
new_list = list(filter(lambda x: (x % 2 == 0), my_list))

print(new_list)
```

### Global variable

	- 함수 외부에 선언된 변수이다.
	- 함수 내부와 외부에서 접근 가능하다.

```python
x = "global"

def foo():
    print("x inside:", x)

foo()
print("x outside:", x)
```

### Local variable

	- 함수 내부에서 선언된 변수이다.
	- 함수 외부에서 사용할 수 없다.

```python
def foo():
    y = "local"
    print(y)

foo()
```

### nonlocal

	- 중첩 함수에서 바깥 함수의 변수를 수정할 때 사용한다.

```python
def outer():
    x = "local"

    def inner():
        nonlocal x
        x = "nonlocal"
        print("inner:", x)

    inner()
    print("outer:", x)

outer()
```
