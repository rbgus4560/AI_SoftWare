## 모듈과 패키지

### 모듈(Module)

	- 명령문과 함수 정의가 들어 있는 `.py` 파일이다.
	- `import`로 가져와 사용할 수 있다.

	예제

```python
import math
print(math.pi)
```

### 모듈 이름 변경

```python
import math as m
print(m.pi)
```

### 특정 이름만 가져오기

```python
from math import pi
print(pi)
```

### 모듈 내부 이름 확인

```python
import math
print(dir(math))
```

### 패키지(Package)

	- 여러 모듈을 폴더 단위로 관리하는 구조이다.
	- 큰 프로젝트를 정리할 때 사용한다.

	예시 구조

```text
my_project/
    main.py
    calculator/
        __init__.py
        add.py
        sub.py
```

### 시험 포인트

	Q. 함수가 자기 자신을 호출하는 방식은?
	정답: 재귀

	Q. 이름 없는 간단한 함수는?
	정답: 람다 함수

	Q. Python 파일 하나를 무엇이라고 하는가?
	정답: 모듈
