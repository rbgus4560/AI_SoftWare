	명령문 (Statement)
		- 프로그램에서 실행되는 기본 단위
```
a = 1
```

	여러 줄 명령문 (Multi-line Statement)
		- 자료형으로 처리 : 괄호 (), 대괄호 [], 중괄호 {}
		- 역슬래시(\)
		- 세미콜론(;) 기호로 다중 명령문 가능
```
a = (1 + 2 + 3 +  4 + 5 + 6 + 7 + 8 + 9)

a = 1 + 2 + 3 + \  
	4 + 5 + 6 + \
	7 + 8 + 9

a = 1; b = 2; c = 3;
```

	들여쓰기 (Indentation)
		- 파이썬은 {} 대신 들여쓰기로 블록을 정의.
		- 일반적으로 4칸을 사용. ※ Tab 사용
```
ex)
for i in range(1,11) :
	print(i)
	if i == 5:
		break
		
	→ 1
	  2 
	  3 
	  4 
	  5
```
