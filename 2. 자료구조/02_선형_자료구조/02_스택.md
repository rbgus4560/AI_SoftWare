## 스택

### 스택(Stack)

	- 후입선출 구조
	- LIFO: Last-In First-Out
	- 마지막에 들어온 데이터가 가장 먼저 나간다.
	- 삽입과 삭제는 스택의 top에서만 수행한다.

	구조

```text
top
 ↓
[30] ← 마지막 삽입, 먼저 삭제
[20]
[10]
```

	실생활 예시

	- 접시 쌓기
	- 뒤로 가기
	- 문서 편집기 되돌리기
	- 함수 호출 관리
	- 괄호 검사
	- 수식 계산
	- 미로 탐색

### 스택 ADT

```text
Push(e)    : 스택 맨 위에 요소 추가
Pop()      : 스택 맨 위 요소 삭제 및 반환
Peek()     : 스택 맨 위 요소 확인
isEmpty()  : 공백 상태 확인
isFull()   : 포화 상태 확인
Size()     : 요소 개수 반환
Clear()    : 스택 초기화
```

### 스택 클래스 구현

```python
class Stack:
    def __init__(self):
        self.items = []

    def isEmpty(self):
        return len(self.items) == 0

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if self.isEmpty():
            return None
        return self.items.pop()

    def peek(self):
        if self.isEmpty():
            return None
        return self.items[-1]

    def size(self):
        return len(self.items)

    def clear(self):
        self.items.clear()


s = Stack()
s.push(10)
s.push(20)

print(s.peek())
print(s.pop())
print(s.size())
```

	실행 결과

```text
20
20
1
```

### 오버플로와 언더플로

	오버플로(Overflow)

	- 스택이 가득 찼는데 삽입하려는 경우

	언더플로(Underflow)

	- 스택이 비어 있는데 삭제하려는 경우

### 스택 응용: 괄호 검사

	핵심 아이디어

	- 여는 괄호는 스택에 넣는다.
	- 닫는 괄호가 나오면 스택에서 하나 꺼낸다.
	- 끝났을 때 스택이 비어 있으면 정상이다.

	예제 코드

```python
def check_bracket(expr):
    stack = []

    for ch in expr:
        if ch in "({[":
            stack.append(ch)

        elif ch in ")}]":
            if not stack:
                return False

            left = stack.pop()

            if ch == ")" and left != "(":
                return False
            if ch == "}" and left != "{":
                return False
            if ch == "]" and left != "[":
                return False

    return len(stack) == 0


print(check_bracket("{ A[(i+1)] = 0 }"))
print(check_bracket("{ A[(i+1]) = 0 }"))
```

	실행 결과

```text
True
False
```

	시험 포인트

	괄호 검사 문제는 거의 항상 스택을 사용한다.

### 스택 응용: 후위 표기식 계산

	표기 방식

| 표기 | 예 |
|---|---|
| 중위 표기 | A + B |
| 전위 표기 | + A B |
| 후위 표기 | A B + |

	후위 표기식 계산 방법

	- 숫자는 스택에 push
	- 연산자를 만나면 숫자 2개를 pop
	- 계산 결과를 다시 push

	예제

```python
def calc_postfix(expr):
    stack = []

    for token in expr.split():
        if token.isdigit():
            stack.append(int(token))
        else:
            b = stack.pop()
            a = stack.pop()

            if token == "+":
                stack.append(a + b)
            elif token == "-":
                stack.append(a - b)
            elif token == "*":
                stack.append(a * b)
            elif token == "/":
                stack.append(a / b)

    return stack.pop()


print(calc_postfix("8 2 / 3 2 * +"))
```

	실행 결과

```text
10.0
```

### 스택 응용: 미로 탐색 DFS

	- DFS는 깊이 우선 탐색이다.
	- 지나온 경로를 스택에 저장한다.
	- 한 방향으로 계속 가다가 막히면 되돌아온다.

	핵심 흐름

```text
1. 시작 위치를 스택에 넣는다.
2. 스택에서 위치를 꺼낸다.
3. 방문 표시를 한다.
4. 출구이면 성공.
5. 주변 위치를 스택에 넣는다.
6. 반복한다.
```
