## 큐와 덱

### 큐(Queue)

	- 선입선출 구조
	- FIFO: First-In First-Out
	- 먼저 들어온 데이터가 먼저 나간다.
	- 삽입은 rear에서, 삭제는 front에서 수행한다.

	구조

```text
삭제(front) ← [A][B][C] ← 삽입(rear)
```

	실생활 예시

	- 은행 대기줄
	- 프린터 작업 큐
	- 콜 큐
	- 버퍼
	- 네트워크 패킷
	- 동영상 스트리밍 버퍼링

---

### 큐 ADT

```text
Enqueue(e) : 큐 뒤에 요소 추가
Dequeue()  : 큐 앞 요소 삭제 및 반환
Peek()     : 큐 앞 요소 확인
isEmpty()  : 큐가 비었는지 확인
isFull()   : 큐가 가득 찼는지 확인
Size()     : 요소 개수 반환
Clear()    : 큐 초기화
```

---

### 파이썬 리스트로 큐 구현

```python
queue = []

queue.append("A")
queue.append("B")
queue.append("C")

print(queue.pop(0))
print(queue.pop(0))
print(queue)
```

	실행 결과

```text
A
B
['C']
```

	주의사항

	`pop(0)`은 앞 요소를 제거한 뒤 나머지 요소를 앞으로 당기므로 비효율적일 수 있다.

---

### collections.deque 사용

```python
from collections import deque

q = deque()

q.append("A")
q.append("B")
q.append("C")

print(q.popleft())
print(q.popleft())
```

	실행 결과

```text
A
B
```

---

### 선형 큐의 문제점

	- 앞에서 삭제가 반복되면 배열 앞부분이 비게 된다.
	- 뒤쪽 공간이 부족해 보일 수 있다.
	- 실제로는 공간이 남아 있는데 삽입이 어려워질 수 있다.

---

### 원형 큐(Circular Queue)

	- 배열의 끝과 처음을 연결한 것처럼 사용하는 큐
	- front와 rear를 나머지 연산으로 이동한다.

	핵심 공식

```text
front = (front + 1) % MAX_SIZE
rear  = (rear + 1) % MAX_SIZE
```

	공백 상태

```text
front == rear
```

	포화 상태

```text
front == (rear + 1) % MAX_SIZE
```

	시험 포인트

	원형 큐는 공백 상태와 포화 상태를 구분하기 위해 보통 한 칸을 비워둔다.

---

### queue 모듈

```python
import queue

Q = queue.Queue(maxsize=20)

for i in range(1, 5):
    Q.put(i)

while not Q.empty():
    print(Q.get())
```

	실행 결과

```text
1
2
3
4
```

	스택처럼 사용

```python
import queue

S = queue.LifoQueue()

S.put(1)
S.put(2)
S.put(3)

print(S.get())
```

	실행 결과

```text
3
```

### 큐 응용: BFS

	- BFS는 너비 우선 탐색이다.
	- 가까운 위치를 먼저 방문한다.
	- 큐를 사용한다.

	핵심 흐름

```text
1. 시작 위치를 큐에 넣는다.
2. 큐에서 위치를 꺼낸다.
3. 방문한다.
4. 주변 위치를 큐에 넣는다.
5. 반복한다.
```

### 덱(Deque)

	- Double Ended Queue
	- 앞과 뒤 모두에서 삽입/삭제 가능
	- 스택처럼도, 큐처럼도 사용할 수 있다.
	- 단, 중간 삽입/삭제를 허용하는 구조는 아니다.

	덱 ADT

```text
AddFront(e)
DeleteFront()
GetFront()
AddRear(e)
DeleteRear()
GetRear()
isEmpty()
isFull()
Size()
Clear()
```

	파이썬 예시

```python
from collections import deque

dq = deque()

dq.appendleft("A")
dq.append("B")
dq.append("C")

print(dq.popleft())
print(dq.pop())
```

	실행 결과

```text
A
C
```

### 우선순위 큐(Priority Queue)

	- 각 데이터가 우선순위를 가진다.
	- 우선순위가 높은 데이터가 먼저 나온다.
	- 운영체제 작업 스케줄링, 네트워크 트래픽 제어 등에 사용된다.

	예시

```python
import heapq

pq = []

heapq.heappush(pq, (2, "일반 작업"))
heapq.heappush(pq, (1, "긴급 작업"))
heapq.heappush(pq, (3, "낮은 우선순위"))

while pq:
    print(heapq.heappop(pq))
```

	실행 결과

```text
(1, '긴급 작업')
(2, '일반 작업')
(3, '낮은 우선순위')
```

	시험 포인트

	파이썬 `heapq`는 기본적으로 최소 힙이다. 숫자가 작을수록 먼저 나온다.
