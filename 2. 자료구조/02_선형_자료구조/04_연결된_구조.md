## 연결된 구조

### 연결된 구조(Linked Structure)

	- 데이터가 메모리에 흩어져 있어도 링크로 연결하여 관리한다.
	- 각 노드는 데이터와 링크를 가진다.

	노드(Node)

```text
[ data | link ]
```

	연결 리스트 구조

```text
head → [10 | next] → [20 | next] → [30 | None]
```

---

### 노드 클래스

```python
class Node:
    def __init__(self, data, link=None):
        self.data = data
        self.link = link
```

---

### 단순 연결 리스트

	- 한 방향으로만 연결된 리스트
	- 각 노드는 다음 노드의 주소만 가진다.

	예시

```python
n1 = Node(10)
n2 = Node(20)
n3 = Node(30)

n1.link = n2
n2.link = n3

cur = n1
while cur is not None:
    print(cur.data)
    cur = cur.link
```

	실행 결과

```text
10
20
30
```

---

### 연결된 스택

	- 노드를 이용해 스택 구현
	- top이 첫 노드를 가리킨다.

	개념

```text
top → [30] → [20] → [10]
```

	장점

	- 크기 제한이 상대적으로 자유롭다.
	- 삽입/삭제가 빠르다.

---

### 원형 연결 리스트

	- 마지막 노드가 첫 번째 노드를 가리킨다.

```text
[10] → [20] → [30]
 ↑             ↓
 └─────────────┘
```

	사용 예

	- 반복 순환 구조
	- 원형 큐
	- 턴 기반 게임

---

### 이중 연결 리스트

	- 각 노드가 이전 노드와 다음 노드 링크를 모두 가진다.

```text
None ← [prev | data | next] ↔ [prev | data | next] → None
```

	장점

	- 양방향 이동 가능

	단점

	- 링크가 2개라 메모리를 더 사용한다.
	- 구현이 복잡하다.
