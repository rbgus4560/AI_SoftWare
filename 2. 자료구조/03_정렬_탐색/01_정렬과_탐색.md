## 정렬과 탐색

### 정렬(Sorting)

	- 데이터를 특정 기준에 따라 순서대로 재배열하는 것
	- 탐색, 집합 연산, 데이터 분석의 기본이 된다.

	정렬 기준

	- 숫자 크기
	- 문자열 사전순
	- 날짜
	- 점수
	- 우선순위

---

### 정렬 관련 용어

```text
레코드: 정렬 대상
키: 정렬 기준이 되는 값
```

	예시

```python
students = [
    {"name": "Kim", "score": 80},
    {"name": "Lee", "score": 90}
]
```

	여기서 `score`가 정렬 키가 될 수 있다.

---

### 선택 정렬(Selection Sort)

	- 리스트에서 가장 작은 값을 찾아 앞쪽으로 옮긴다.
	- 시간 복잡도: `O(n^2)`

	예제 코드

```python
def selection_sort(A):
    n = len(A)

    for i in range(n - 1):
        least = i

        for j in range(i + 1, n):
            if A[j] < A[least]:
                least = j

        A[i], A[least] = A[least], A[i]


arr = [5, 3, 8, 1, 2]
selection_sort(arr)
print(arr)
```

	실행 결과

```text
[1, 2, 3, 5, 8]
```

	시험 포인트

	선택 정렬은 매 단계마다 최솟값을 선택한다.

---

### 삽입 정렬(Insertion Sort)

	- 정렬된 부분에 새 값을 알맞은 위치에 삽입한다.
	- 이미 어느 정도 정렬된 데이터에 효율적이다.

	예제 코드

```python
def insertion_sort(A):
    for i in range(1, len(A)):
        key = A[i]
        j = i - 1

        while j >= 0 and A[j] > key:
            A[j + 1] = A[j]
            j -= 1

        A[j + 1] = key


arr = [5, 3, 8, 1, 2]
insertion_sort(arr)
print(arr)
```

	실행 결과

```text
[1, 2, 3, 5, 8]
```

---

### 버블 정렬(Bubble Sort)

	- 인접한 두 값을 비교하여 순서가 잘못되어 있으면 교환한다.
	- 큰 값이 뒤쪽으로 거품처럼 이동한다.

	예제 코드

```python
def bubble_sort(A):
    n = len(A)

    for i in range(n - 1):
        for j in range(n - 1 - i):
            if A[j] > A[j + 1]:
                A[j], A[j + 1] = A[j + 1], A[j]


arr = [5, 3, 8, 1, 2]
bubble_sort(arr)
print(arr)
```

	실행 결과

```text
[1, 2, 3, 5, 8]
```

### 탐색(Search)

	- 자료 중 원하는 값을 찾는 작업
	- 상품 찾기, 단어 검색, 회원 검색 등에 사용된다.

---

### 순차 탐색(Sequential Search)

	- 처음부터 끝까지 하나씩 비교한다.
	- 정렬되지 않은 데이터에도 적용 가능하다.
	- 시간 복잡도: `O(n)`

	예제 코드

```python
def sequential_search(A, key):
    for i in range(len(A)):
        if A[i] == key:
            return i
    return -1


arr = [10, 20, 30, 40]
print(sequential_search(arr, 30))
```

	실행 결과

```text
2
```

---

### 이진 탐색(Binary Search)

	- 정렬된 데이터에서 사용한다.
	- 가운데 값을 기준으로 탐색 범위를 절반씩 줄인다.
	- 시간 복잡도: `O(log n)`

	예제 코드

```python
def binary_search(A, key):
    low = 0
    high = len(A) - 1

    while low <= high:
        mid = (low + high) // 2

        if A[mid] == key:
            return mid
        elif A[mid] < key:
            low = mid + 1
        else:
            high = mid - 1

    return -1


arr = [10, 20, 30, 40, 50]
print(binary_search(arr, 40))
```

	실행 결과

```text
3
```

	시험 포인트

	이진 탐색은 반드시 정렬된 데이터에서 사용한다.

---

### 보간 탐색(Interpolation Search)

	- 이진 탐색의 변형
	- 탐색 키가 있을 위치를 예측하여 탐색한다.
	- 데이터가 균등하게 분포할 때 효과적이다.

---

### 해싱(Hashing)

	- 키 값을 해시 함수에 넣어 저장 위치를 계산한다.
	- 빠른 탐색을 위해 사용된다.

	개념

```text
key → hash function → address
```

	충돌(Collision)

	서로 다른 키가 같은 주소로 계산되는 상황

```text
hash(45) = 5
hash(25) = 5
```

---

### 충돌 해결 방법

	선형 조사법

	- 충돌이 발생하면 다음 칸을 확인한다.

```text
h(key), h(key)+1, h(key)+2 ...
```

	이차 조사법

	- 일정 간격을 제곱 형태로 증가시키며 확인한다.

	이중 해싱

	- 두 번째 해시 함수를 이용한다.

	체이닝

	- 하나의 버킷에 여러 데이터를 연결 리스트로 저장한다.

### 맵(Map)

```text
key → value
```

	예시

