## 고급 정렬

### 셸 정렬(Shell Sort)

	- 삽입 정렬을 개선한 정렬
	- 일정 간격(gap)을 두고 부분적으로 정렬한 뒤 간격을 줄여간다.

	핵심

```text
큰 gap으로 대략 정렬 → gap 감소 → 최종 삽입 정렬
```

---

### 힙 정렬(Heap Sort)

	- 힙 자료구조를 이용한 정렬
	- 최댓값 또는 최솟값을 쉽게 꺼낼 수 있다.

	파이썬 예시

```python
import heapq

arr = [5, 3, 8, 1, 2]
heapq.heapify(arr)

sorted_arr = []

while arr:
    sorted_arr.append(heapq.heappop(arr))

print(sorted_arr)
```

	실행 결과

```text
[1, 2, 3, 5, 8]
```

### 병합 정렬(Merge Sort)

	- 분할 정복 방식의 정렬
	- 리스트를 반으로 나누고, 각각 정렬한 뒤 병합한다.
	- 시간 복잡도: `O(n log n)`

	예제

```python
def merge_sort(A):
    if len(A) <= 1:
        return A

    mid = len(A) // 2
    left = merge_sort(A[:mid])
    right = merge_sort(A[mid:])

    return merge(left, right)


def merge(left, right):
    result = []
    i = j = 0

    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result.extend(left[i:])
    result.extend(right[j:])
    return result


print(merge_sort([5, 3, 8, 1, 2]))
```

	실행 결과

```text
[1, 2, 3, 5, 8]
```

### 퀵 정렬(Quick Sort)

	- 분할 정복 방식
	- 피벗을 기준으로 작은 값과 큰 값을 나눈다.
	- 평균 시간 복잡도: `O(n log n)`
	- 최악 시간 복잡도: `O(n^2)`

	예제

```python
def quick_sort(A):
    if len(A) <= 1:
        return A

    pivot = A[0]
    left = [x for x in A[1:] if x <= pivot]
    right = [x for x in A[1:] if x > pivot]

    return quick_sort(left) + [pivot] + quick_sort(right)


print(quick_sort([5, 3, 8, 1, 2]))
```

	실행 결과

```text
[1, 2, 3, 5, 8]
```

### 듀얼 피벗 퀵 정렬

	- 피벗을 2개 사용하는 퀵 정렬
	- 일반 퀵 정렬보다 성능이 좋다고 알려져 있다.
	- 구현은 일반 퀵 정렬보다 복잡하다.

### 기수 정렬(Radix Sort)

	- 숫자의 각 자리수를 기준으로 단계적으로 정렬한다.
	- 비교 기반 정렬이 아니다.
	- 추가 메모리가 필요하다.
	- 시간 복잡도: `O(k n)`

	핵심

```text
1의 자리 정렬
10의 자리 정렬
100의 자리 정렬
...
```
