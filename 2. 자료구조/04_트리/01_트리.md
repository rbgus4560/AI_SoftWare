## 트리

### 트리(Tree)

	- 계층적인 자료 표현에 적합한 자료구조
	- 조직도, 폴더 구조, 가계도 등에 사용된다.

	구조

```text
        A
      /   \
     B     C
    / \
   D   E
```

---

### 트리 용어

| 용어 | 의미 |
|---|---|
| 루트 | 가장 위 노드 |
| 부모 노드 | 자식을 가진 노드 |
| 자식 노드 | 부모 아래 노드 |
| 단말 노드 | 자식이 없는 노드 |
| 간선 | 노드를 연결하는 선 |
| 레벨 | 루트로부터의 단계 |
| 높이 | 가장 깊은 레벨 |

---

### 이진 트리(Binary Tree)

	- 각 노드가 최대 2개의 자식을 가지는 트리
	- 왼쪽 서브트리와 오른쪽 서브트리로 구성된다.

	예시

```text
      A
    /   \
   B     C
```

---

### 이진 트리 종류

	포화 이진 트리

	- 모든 레벨에 노드가 꽉 차 있는 트리

	완전 이진 트리

	- 마지막 레벨을 제외하고 꽉 차 있으며, 마지막 레벨은 왼쪽부터 채워진 트리

---

### 이진 트리 노드 구현

```python
class TNode:
    def __init__(self, data, left=None, right=None):
        self.data = data
        self.left = left
        self.right = right
```

---

### 이진 트리 순회

	트리에 있는 모든 노드를 한 번씩 방문하는 작업이다.

	전위 순회(Preorder)

```text
루트 → 왼쪽 → 오른쪽
```

	중위 순회(Inorder)

```text
왼쪽 → 루트 → 오른쪽
```

	후위 순회(Postorder)

```text
왼쪽 → 오른쪽 → 루트
```

	예제 코드

```python
def preorder(n):
    if n is not None:
        print(n.data, end=" ")
        preorder(n.left)
        preorder(n.right)

def inorder(n):
    if n is not None:
        inorder(n.left)
        print(n.data, end=" ")
        inorder(n.right)

def postorder(n):
    if n is not None:
        postorder(n.left)
        postorder(n.right)
        print(n.data, end=" ")


root = TNode("A", TNode("B"), TNode("C"))

preorder(root)
print()
inorder(root)
print()
postorder(root)
```

	실행 결과

```text
A B C
B A C
B C A
```

### 레벨 순회

	- 레벨 순서대로 방문한다.
	- 큐를 사용한다.

```python
from collections import deque

def levelorder(root):
    q = deque()

    if root is not None:
        q.append(root)

    while q:
        n = q.popleft()
        print(n.data, end=" ")

        if n.left:
            q.append(n.left)
        if n.right:
            q.append(n.right)
```

### 힙(Heap)

	- 완전 이진 트리 기반 자료구조
	- 우선순위 큐 구현에 많이 사용된다.

	최대 힙

	- 부모 노드가 자식 노드보다 크거나 같다.

	최소 힙

	- 부모 노드가 자식 노드보다 작거나 같다.

	파이썬 heapq

```python
import heapq

heap = []

heapq.heappush(heap, 5)
heapq.heappush(heap, 2)
heapq.heappush(heap, 8)

print(heapq.heappop(heap))
```

	실행 결과

```text
2
```

	시험 포인트

	파이썬 `heapq`는 최소 힙이다.
