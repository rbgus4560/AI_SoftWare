## 탐색트리

### 이진 탐색 트리(BST)

	- 탐색을 빠르게 하기 위한 이진 트리
	- 왼쪽 서브트리는 루트보다 작은 값
	- 오른쪽 서브트리는 루트보다 큰 값

	구조

```text
        50
      /    \
    30      70
   /  \    /  \
 20   40  60  80
```

---

### BST 탐색

```python
def search(root, key):
    if root is None:
        return None

    if key == root.data:
        return root
    elif key < root.data:
        return search(root.left, key)
    else:
        return search(root.right, key)
```

	시간 복잡도

	- 균형이 잘 잡힌 경우: `O(log n)`
	- 한쪽으로 치우친 경우: `O(n)`

---

### BST 삽입

```python
def insert(root, key):
    if root is None:
        return TNode(key)

    if key < root.data:
        root.left = insert(root.left, key)
    elif key > root.data:
        root.right = insert(root.right, key)

    return root
```

### BST 삭제

	삭제는 가장 복잡한 연산이다.

	삭제 경우

| 경우 | 설명 |
|---|---|
| Case 1 | 단말 노드 삭제 |
| Case 2 | 자식이 하나인 노드 삭제 |
| Case 3 | 자식이 둘인 노드 삭제 |

	Case 1: 단말 노드

	그냥 삭제하면 된다.

	Case 2: 자식이 하나

	삭제 노드의 부모가 삭제 노드의 자식을 가리키게 한다.

	Case 3: 자식이 둘

	후계 노드 또는 선행 노드를 찾아 삭제 위치로 이동한다.

### AVL 트리

	- 균형 있는 이진 탐색 트리
	- 탐색, 삽입, 삭제를 `O(log n)`으로 유지한다.

	균형 인수

```text
balance factor = 왼쪽 서브트리 높이 - 오른쪽 서브트리 높이
```

	보통 균형 인수가 -1, 0, 1이면 균형 상태이다.

---

### AVL 회전

	LL 회전

	- 왼쪽-왼쪽 방향으로 불균형
	- 오른쪽 회전

	RR 회전

	- 오른쪽-오른쪽 방향으로 불균형
	- 왼쪽 회전

	LR 회전

	- 왼쪽-오른쪽 방향 불균형
	- 왼쪽 회전 후 오른쪽 회전

	RL 회전

	- 오른쪽-왼쪽 방향 불균형
	- 오른쪽 회전 후 왼쪽 회전

	시험 포인트

	AVL 트리는 BST의 최악 성능을 개선하기 위해 균형을 유지한다.
