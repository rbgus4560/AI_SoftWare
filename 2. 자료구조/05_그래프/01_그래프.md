## 그래프

### 그래프(Graph)

	- 객체들 사이의 관계를 표현하는 자료구조
	- 정점과 간선으로 구성된다.

```text
G = (V, E)
V: 정점 집합
E: 간선 집합
```

	예시

```text
A --- B
|     |
C --- D
```

---

### 그래프 용어

| 용어 | 의미 |
|---|---|
| 정점(Vertex) | 객체 |
| 간선(Edge) | 정점 사이의 연결 |
| 인접 정점 | 간선으로 직접 연결된 정점 |
| 경로 | 정점을 따라 이동하는 길 |
| 차수 | 정점에 연결된 간선 수 |
| 연결 그래프 | 모든 정점 사이에 경로 존재 |
| 사이클 | 시작점으로 다시 돌아오는 경로 |

---

### 그래프 종류

	무방향 그래프

	- 간선에 방향이 없다.

```text
A --- B
```

	방향 그래프

	- 간선에 방향이 있다.

```text
A → B
```

	가중치 그래프

	- 간선에 비용 또는 거리 정보가 있다.

```text
A --5-- B
```

---

### 그래프 표현 방법

### 인접 행렬

	- 2차원 배열로 연결 관계를 표현한다.

```python
graph = [
    [0, 1, 1],
    [1, 0, 0],
    [1, 0, 0]
]
```

	장점

	- 간선 존재 여부 확인이 빠르다.

	단점

	- 정점 수가 많으면 메모리를 많이 사용한다.

---

### 인접 리스트

	- 각 정점마다 연결된 정점 목록을 저장한다.

```python
graph = {
    "A": ["B", "C"],
    "B": ["A"],
    "C": ["A"]
}
```

	장점

	- 메모리 효율적이다.
	- 간선이 적은 그래프에 적합하다.

---

### DFS

	- Depth First Search
	- 깊이 우선 탐색
	- 스택 또는 재귀 사용

	예제

```python
def dfs(graph, v, visited):
    visited.add(v)
    print(v, end=" ")

    for nxt in graph[v]:
        if nxt not in visited:
            dfs(graph, nxt, visited)


graph = {
    "A": ["B", "C"],
    "B": ["D"],
    "C": [],
    "D": []
}

dfs(graph, "A", set())
```

	실행 결과

```text
A B D C
```

---

### BFS

	- Breadth First Search
	- 너비 우선 탐색
	- 큐 사용

	예제

```python
from collections import deque

def bfs(graph, start):
    visited = set([start])
    q = deque([start])

    while q:
        v = q.popleft()
        print(v, end=" ")

        for nxt in graph[v]:
            if nxt not in visited:
                visited.add(nxt)
                q.append(nxt)


graph = {
    "A": ["B", "C"],
    "B": ["D"],
    "C": [],
    "D": []
}

bfs(graph, "A")
```

	실행 결과

```text
A B C D
```

	DFS vs BFS

| 구분 | DFS | BFS |
|---|---|---|
| 방식 | 깊게 탐색 | 넓게 탐색 |
| 자료구조 | 스택/재귀 | 큐 |
| 예 | 미로 경로 탐색 | 최단 단계 탐색 |

---

### 연결 성분

	- 최대로 연결된 부분 그래프
	- 그래프가 여러 덩어리로 나뉘어 있을 때 각 덩어리를 연결 성분이라고 한다.

---

### 신장 트리

	- 그래프의 모든 정점을 포함하는 트리
	- 사이클이 없어야 한다.
	- 정점이 `n`개라면 간선은 `n-1`개이다.

---

### 위상 정렬

	- 방향 그래프에서 선행 관계를 지키며 정점을 나열하는 것
	- 사이클이 있으면 위상 정렬이 불가능하다.

	예시

```text
선수 과목 → 다음 과목
자료구조 → 알고리즘 → 프로젝트
```

	핵심 과정

```text
1. 진입 차수가 0인 정점 선택
2. 해당 정점과 연결된 간선 제거
3. 다시 진입 차수가 0인 정점 선택
4. 반복
```
