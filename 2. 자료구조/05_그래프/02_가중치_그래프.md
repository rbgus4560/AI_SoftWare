## 가중치 그래프

### 가중치 그래프

	- 간선에 비용, 거리, 시간 등의 값이 붙은 그래프

```text
A --3-- B
B --5-- C
```

	사용 예

	- 지도 경로
	- 네트워크 비용
	- 배송 비용
	- 최소 시간 경로

---

### 최소 비용 신장 트리(MST)

	- 모든 정점을 연결하면서 간선 비용 합이 최소인 신장 트리
	- 사이클이 없어야 한다.

---

### Kruskal 알고리즘

	- 간선을 비용이 작은 순서대로 선택한다.
	- 사이클이 생기면 선택하지 않는다.
	- Union-Find를 사용해 사이클을 검사한다.

	과정

```text
1. 모든 간선을 비용순으로 정렬
2. 가장 작은 간선부터 선택
3. 사이클이 생기면 제외
4. 간선 수가 n-1개가 될 때까지 반복
```

	간단 예제

```python
parent = {}

def find(x):
    if parent[x] != x:
        parent[x] = find(parent[x])
    return parent[x]

def union(a, b):
    root_a = find(a)
    root_b = find(b)

    if root_a != root_b:
        parent[root_b] = root_a
        return True
    return False


vertices = ["A", "B", "C"]
edges = [
    (1, "A", "B"),
    (3, "A", "C"),
    (2, "B", "C")
]

for v in vertices:
    parent[v] = v

edges.sort()

mst = []
for cost, a, b in edges:
    if union(a, b):
        mst.append((a, b, cost))

print(mst)
```

	실행 결과

```text
[('A', 'B', 1), ('B', 'C', 2)]
```

### Prim 알고리즘

	- 하나의 정점에서 시작해서 트리를 점점 확장한다.
	- 현재 트리와 연결된 간선 중 가장 비용이 작은 간선을 선택한다.

	Kruskal vs Prim

| 구분 | Kruskal | Prim |
|---|---|---|
| 선택 기준 | 전체 간선 중 최소 | 현재 트리와 연결된 최소 간선 |
| 핵심 자료구조 | Union-Find | 우선순위 큐 |
| 관점 | 간선 중심 | 정점 중심 |

### 최단 경로

	- 가중치 그래프에서 비용 합이 최소인 경로를 찾는 문제

### Dijkstra 알고리즘

	- 하나의 시작 정점에서 다른 모든 정점까지 최단 경로를 찾는다.
	- 음수 가중치가 없는 그래프에서 사용한다.

	핵심 아이디어

```text
1. 시작 정점의 거리를 0으로 설정
2. 아직 방문하지 않은 정점 중 거리가 가장 짧은 정점 선택
3. 해당 정점을 거쳐 가는 거리로 갱신
4. 반복
```

	예제

```python
import heapq

def dijkstra(graph, start):
    dist = {v: float("inf") for v in graph}
    dist[start] = 0

    pq = [(0, start)]

    while pq:
        cur_dist, cur = heapq.heappop(pq)

        if cur_dist > dist[cur]:
            continue

        for nxt, weight in graph[cur]:
            new_dist = cur_dist + weight

            if new_dist < dist[nxt]:
                dist[nxt] = new_dist
                heapq.heappush(pq, (new_dist, nxt))

    return dist


graph = {
    "A": [("B", 2), ("C", 5)],
    "B": [("C", 1)],
    "C": []
}

print(dijkstra(graph, "A"))
```

	실행 결과

```text
{'A': 0, 'B': 2, 'C': 3}
```

### Floyd-Warshall 알고리즘

	- 모든 정점 사이의 최단 경로를 찾는다.
	- 3중 반복문 사용
	- 시간 복잡도: `O(n^3)`

	핵심 점화식

```text
dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
```

	예제

```python
INF = 999999

dist = [
    [0,   2,   INF],
    [INF, 0,   3],
    [4,   INF, 0]
]

n = len(dist)

for k in range(n):
    for i in range(n):
        for j in range(n):
            dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])

print(dist)
```
