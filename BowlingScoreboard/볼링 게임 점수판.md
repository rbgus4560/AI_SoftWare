
---
### 1. 프로그램 소개
- Python 과 Flet framework 를 활용해 실제 볼링장의 점수판을 구현한 **데스크톱 애플리케이션**입니다. 스트라이크나 스페어처럼 '다음 투구 결과에 따라 점수가 결정되는' 볼링  규칙을 코드로 풀어냈습니다. 코드를 하나에 몰아넣지 않고 '점수 계산', '기호 변환', '화면 구성'의 3가지 역할로 파일을 모듈화 하여 코드를 훨씬 깔끔하고 수정하기 쉽게 설계하였습니다.

---

### 2. 사용언어 및 라이브러리

- **Language:** Python(v3.12.10)
- **GUI Framework:** Flet (`flet` v0.84+)

---

### 3. 시스템 구조

- **`game_engine.py` (비즈니스 로직 / Model)** 
	→ 볼링의 규칙과 점수 연산만을 전담하는 모듈.
    
- **`ui_formatter.py` (데이터 어댑터 / Utility)**
	→ 연산된 숫자 데이터를 UI 렌더링에 적합한 특수 기호 문자열로 바꿔주는 모듈.
    
- **`main.py` (화면 렌더링 및 이벤트 제어 / View & Controller)** 
	→ Flet 프레임워크 기반의 화면 구성 및 사용자 이벤트 처리.
    

---

### 4. 모듈

### [Module 1] `game_engine.py`

- **`class BowlingGame`**
    
    - **상태 변수
        
        - `rolls (list)`: 사용자가 굴린 매 투구의 쓰러뜨린 핀 개수를 순차적으로 누적하는 배열.
            
        - `current_frame (int)`: 현재 진행 중인 프레임 번호 (1 ~ 10).
            
        - `throw_in_frame (int)`: 현재 프레임 내 투구 차수 (1 ~ 3).
            
        - `pins_standing (int)`: 현재 레인에 서 있는 핀의 개수. 매 프레임 혹은 보너스 투구 시 10으로 초기화됨.
            
        - `is_game_over (bool)`: 게임 종료 상태를 제어하는 플래그.
            
    - **메서드 (Methods): `roll(pins)`
        
        - **역할:** 쓰러진 핀 개수의 결과에 따라 게임의 순서를 다음 단계로 맞게 이동시킨다.
            
    - **메서드 (Methods): `calculate_scores()`
        
        - **역할:** `rolls` 배열을 순회하며 프레임별 누적 점수를 계산하여 리스트로 반환한다.

### [Module 2] `ui_formatter.py`

- **함수 (Function): `get_frame_displays(rolls)`
    
    - **역할:** 정수형 배열을 UI 렌더링에 적합한 볼링 전용 기호 배열로 변환합니다.
        

### [Module 3] `main.py`

- **함수 (Function): `update_scoreboard_ui()`
    
    - **역할:** 로직 연산 결과와 포매팅된 기호 데이터를 바탕으로 점수판을 동적으로 다시 그립니다.

        
- **함수 (Function): `on_roll_click(e)`
    
    - **역할:** 사용자가 버튼을 누르면 랜덤으로 점수를 정하고, 그 숫자를 계산기에게 전달해 주는 역할을 합니다.
